-----ИНСТРУКЦИЯ-----

1. Произвести инсталяцию требований(они же зависимости, они же требования):
	pip install -r requirements.txt (в директории с программой находится этот файл).
2. Запустить chat_server.py (можно ввести значения порта и хоста в строке запуска, 
	если этого не сделать, запуск произойдёт на 192.168.111.1 2552).
3. Запустить chat client.py (можно ввести значения порта и хоста в строке запуска, 
	если этого не сделать, запуск произойдёт на 192.168.111.1 2552).
4. Открывается интерфейс чата , в окне сообщений будут дальнейшие инструкции.

*в данной версии использование команды "/хроника" ведёт к крашу клиента
*можно создать множество серверов и присоединяться к ним с клиента
